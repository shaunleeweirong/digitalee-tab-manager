# Extension Icons

Place your extension icons in this directory:

- `icon-16.png` - 16x16px (toolbar)
- `icon-32.png` - 32x32px (Windows)
- `icon-48.png` - 48x48px (extension management)
- `icon-128.png` - 128x128px (Chrome Web Store)

For now, you can use placeholder icons or generate them using a tool like:
- https://www.figma.com
- https://www.canva.com
- Or use a simple solid color square as a placeholder

The icons should follow the design aesthetic outlined in the PRD:
- Minimal, clean design
- Blue accent color (#3B82F6 for light mode, #60A5FA for dark mode)
- Simple folder or tab icon metaphor

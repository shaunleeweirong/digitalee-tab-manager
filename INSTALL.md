# How to Install Digitalee Tab Manager

**A step-by-step guide for installing the Digitalee Tab Manager Chrome extension.**

---

## 📋 What You Need

- ✅ Google Chrome browser (any recent version)
- ✅ The downloaded ZIP file: `digitalee-tab-manager-v*.zip`
- ✅ 5 minutes of your time

**No technical knowledge required!** Just follow the pictures and instructions below.

---

## 🚀 Installation Steps

### Step 1: Extract the ZIP File

1. **Find your downloaded ZIP file** (usually in your Downloads folder)
2. **Right-click** on `digitalee-tab-manager-v*.zip`
3. **Select "Extract All"** (Windows) or **double-click** (Mac)
4. **Choose a location** to extract (Desktop is fine)
5. **Click "Extract"**

You should now see a folder called `digitalee-tab-manager` containing the extension files.

### Step 2: Open Chrome Extension Settings

1. **Open Google Chrome**
2. **Type this in the address bar**: `chrome://extensions/`
3. **Press Enter**

**Alternative method:**
- Click the Chrome menu (⋮) in the top-right corner
- Go to **More tools** → **Extensions**

### Step 3: Enable Developer Mode

⚠️ **Don't worry!** This is safe and only allows you to install extensions from files.

1. Look for **"Developer mode"** toggle in the **top-right corner**
2. **Click the toggle** to turn it **ON** (it should turn blue)
3. You'll see new buttons appear: "Load unpacked", "Pack extension", etc.

### Step 4: Load the Extension

1. **Click "Load unpacked"** button
2. **Navigate to the folder** you extracted in Step 1
3. **Select the `digitalee-tab-manager` folder** (the one with manifest.json inside)
4. **Click "Select Folder"** (Windows) or **"Open"** (Mac)

### Step 5: Success! 🎉

✅ **The extension is now installed!**

You should see:
- "Digitalee Tab Manager" appears in your extensions list
- A small icon in your Chrome toolbar
- Chrome shows a warning banner about developer mode (this is normal and safe)

### Step 6: Test the Extension

1. **Open a new tab** (Cmd+T or Ctrl+T)
2. **You should see** the Digitalee Tab Manager instead of Chrome's default new tab page
3. **Try saving a tab** by clicking the save button in the sidebar

---

## ⚠️ Important Notes

### About Security Warnings

Chrome will show warnings like:
- **"Disable developer mode extensions"**
- **"Extensions running in developer mode"**

**This is normal and safe!** These warnings appear because the extension isn't from the Chrome Web Store. The extension is completely safe to use.

### If Chrome Disables the Extension

Sometimes Chrome may disable developer mode extensions after updates:

1. Go back to `chrome://extensions/`
2. Turn **Developer mode ON** again
3. The extension will work normally

---

## 🔧 Troubleshooting

### Problem: "Load unpacked" button is missing
**Solution:** Make sure Developer mode is turned ON (Step 3)

### Problem: "Could not load extension"
**Solutions:**
- Make sure you selected the `digitalee-tab-manager` folder (not the ZIP file)
- The folder should contain `manifest.json` file
- Try extracting the ZIP file again

### Problem: Extension doesn't appear
**Solutions:**
- Refresh the extensions page (F5)
- Check that the extension is enabled (toggle should be blue)
- Try restarting Chrome

### Problem: New tab page doesn't change
**Solutions:**
- Open a completely new tab (not refresh current one)
- Check the extension is enabled in `chrome://extensions/`
- Try disabling other new tab extensions

### Problem: Chrome keeps showing warnings
**This is normal behavior.** The warnings are harmless and don't affect the extension's functionality.

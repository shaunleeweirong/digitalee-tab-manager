# Quick Start Guide

**Get started with Digitalee Tab Manager in 5 minutes!**

---

## 🎯 What is Digitalee Tab Manager?

Digitalee Tab Manager replaces your Chrome new tab page with a powerful workspace for organizing and finding your browser tabs. Think of it as a visual bookmark manager that helps you save, organize, and quickly find any webpage you've visited.

---

## ✨ Key Features at a Glance

- 📂 **Collections** - Organize tabs into named groups
- 🔍 **Smart Search** - Find any saved tab instantly
- 💾 **Save All Tabs** - Capture entire browsing sessions
- 🎨 **Clean Interface** - Minimal, Apple-inspired design
- 🔄 **Real-time Updates** - See open tabs and save them instantly

---

## 🚀 First Time Setup

### 1. Open a New Tab
After installation, open a new tab (Cmd+T or Ctrl+T). You'll see the Digitalee Tab Manager interface instead of Chrome's default page.

### 2. Tour the Interface

**Main Area (Center):**
- Collections grid showing your saved tab collections
- Search bar at the top
- Empty state when you first start

**Sidebar (Right):**
- Currently open tabs in this window
- Save buttons for individual tabs
- "Save All Tabs" button at the top

---

## 📖 Basic Usage

### Saving Your First Tab

1. **Open some websites** in different tabs
2. **Open a new tab** to see Digitalee Tab Manager
3. **Look at the sidebar** - you'll see all your open tabs
4. **Click the save button** (💾) next to any tab
5. **Choose "Create New Collection"**
6. **Give it a name** like "Work Resources" or "Recipe Ideas"
7. **Click Save** - your first collection is created!

### Saving Multiple Tabs at Once

1. **Open several related tabs** (e.g., research articles, shopping items)
2. **Open a new tab** to access Digitalee Tab Manager
3. **Click "Save All Tabs"** at the top of the sidebar
4. **Enter a collection name** like "Christmas Shopping" or "Project Research"
5. **Click Save** - all open tabs are saved as one collection!

### Finding Saved Tabs

1. **Use the search bar** at the top
2. **Type any word** from the website title or URL
3. **See instant results** with matching tabs highlighted
4. **Click any result** to open that webpage immediately

### Managing Collections

**Rename a Collection:**
- Click the collection title and start typing

**Delete a Collection:**
- Click the menu button (⋯) on any collection
- Select "Delete Collection"
- Confirm the deletion

**Add More Tabs to Existing Collections:**
- Save a tab and choose an existing collection from the dropdown

## 🎨 Understanding the Interface

### Collection Cards
Each collection appears as a card showing:
- 📁 **Collection name** (editable)
- 🔢 **Number of saved tabs**
- 👁️ **First few tabs** as previews with favicons
- ⋯ **Menu button** for actions (rename, delete)

### Sidebar Features
- **Green checkmarks** = Tab is already saved
- **Save buttons** = Click to save unsaved tabs
- **Collapse/expand** = Toggle sidebar visibility
- **Show more/less** = View all tabs when you have many open

### Search Results
- **Collections** and **Individual URLs** shown separately
- **Highlighted text** shows why each result matched
- **Click collection names** to scroll to that collection
- **Click URLs** to open them immediately

## 💡 Pro Tips

### Organizing Your Collections
- **Use descriptive names**: "Work - Q4 Planning" instead of "Work Stuff"
- **Create topic-based collections**: "Recipes", "Travel Ideas", "Shopping"
- **Use project-based collections**: "House Renovation", "Wedding Planning"

### Effective Searching
- **Search by website name**: "youtube", "amazon", "reddit"
- **Search by topic**: "recipe", "vacation", "tutorial"
- **Search by partial words**: "goo" finds "Google"
- **Recent tabs appear** when search box is empty

### Workflow Ideas
- **Daily collections**: Save each day's research as dated collections
- **Project collections**: One collection per work/personal project
- **Topic collections**: Ongoing interests like "Cooking" or "Tech News"
- **Session collections**: Save browsing sessions before closing Chrome

## 🔧 Common Questions

**Q: Where are my saved tabs stored?**
A: Locally in Chrome's storage. They don't leave your computer and work offline.

**Q: Can I save the same tab to multiple collections?**
A: Yes! Tabs can belong to multiple collections. Use the dropdown menu on saved tabs.

**Q: What happens if I delete a collection?**
A: The collection and its organization are removed, but the original websites are still accessible via their URLs.

**Q: Why do I see a "developer mode" warning?**
A: This is normal for extensions not installed from the Chrome Web Store. The extension is completely safe.

**Q: Can I export my collections?**
A: Not currently, but this feature is planned for future updates.

**Q: Does this slow down Chrome?**
A: No, the extension is lightweight and only runs when you open new tabs.
